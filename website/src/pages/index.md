## Technologies Used

This book showcases a technology stack consisting of:

- **ROS 2**: Middleware for robot control and communication
- **Gazebo**: Physics simulation environment
- **Unity**: High-fidelity rendering for human-robot interaction
- **NVIDIA Isaac**: Advanced perception and training systems
- **OpenAI APIs**: Voice-to-action systems using Whisper and language models
- **Docusaurus**: Documentation framework with embedded chatbot
